Safari Window Switcher Settings

Customize your window management preferences and keyboard shortcuts.

General Options

When enabled, prepends [Window Name] to active tab titles so the window name shows in Safari's native title bar and tab bar.

Data Management

Wipes all stored custom window labels and session restoration registry clean.

Keyboard Shortcuts

Default hotkey to open Window Switcher: Control + Option + O (^ ⌥ O)

How to Customize Keyboard Shortcuts in macOS:

  1. Open Safari on your Mac.
  2. In the menu bar, go to Safari > Settings... (or press Cmd + ,).
  3. Click the Extensions tab and select Window Switcher.
  4. Click Shortcuts... to assign your preferred key combination (e.g. Control + Option + O or Option + Shift + W).

Security & Privacy Guarantee

  • 🛡️ 100% Local Storage: All window labels are saved locally on your Mac using browser.storage.local.
  • 🔒 Zero Network Tracking: This extension makes zero external HTTP/HTTPS requests.
  • ⚡ Strict CSP Protection: Runs with strict Content Security Policy preventing remote script execution.
